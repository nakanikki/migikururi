# 右くるり (MigiKururi)

右クリックで放射状のパイメニューを開く、Windows 用ランチャーです。
対象アプリの上で右クリックすると、登録したキー操作やアクションを
マウスの方向だけで素早く実行できます（AutoHotPie 系のツール）。

## 特長

- **右クリックで放射パイメニュー** — 対象アプリ上の右クリックを乗っ取って表示
- **方向で選ぶ即時アクション** — カーソルを向けるだけで発動（最速）
- **サブメニュー（多階層）** — セグメントから子・孫メニューへ
- **クイックスロット** — 右クリック押しながら 左/中クリック・ホイールで追加操作
- **アプリごとのプロファイル** — アプリ別にメニューを切り替え
- **直感的な設定エディタ** — ブロックを線で繋いでアクションを組む
- 回転・不透明度・外周/内径サイズなど見た目も細かく調整可能

## ダウンロード・インストール

1. [Releases](https://github.com/nakanikki/migikururi/releases) から最新の `MigiKururi.exe` をダウンロード
2. ダブルクリックで起動（**インストール不要のポータブル版**）
3. 起動すると設定ウィンドウが開きます。対象アプリを指定してメニューを作成してください
4. 終了はタスクトレイのアイコンを右クリック →「終了」

> 設定はユーザーフォルダ（AppData）に保存されます。アンインストールは exe を削除するだけです。

### ⚠ 初回起動時の警告について

署名なしのため、初回起動時に Windows SmartScreen の警告
「**Windows によって PC が保護されました**」が出ることがあります。

- **［詳細情報］** をクリック → **［実行］** で起動できます。

また、起動には **Microsoft Edge WebView2 ランタイム** が必要です
（Windows 10/11 には通常プリインストールされています。無い場合は
[Microsoft の配布ページ](https://developer.microsoft.com/microsoft-edge/webview2/) から導入してください）。

## 使い方（かんたん）

1. 設定ウィンドウで **対象アプリを指定**（キャンバスを右クリック →「対象アプリを追加」→ 起動中のアプリをクリックで取得）
2. パイの各セグメントから **外へドラッグ** してアクション（キー）を設定
3. 対象アプリ上で **右クリック** → パイメニューが出る → 方向を選んで離す

タブを右クリックすると「プロファイルを複製」もできます。

## 開発・ビルド

[Tauri](https://tauri.app/)（Rust + Vanilla JS）製です。

```sh
npm install
npm run tauri dev     # 開発（ホットリロード）
npm run tauri build   # 本番ビルド → src-tauri/target/release/MigiKururi.exe
```
