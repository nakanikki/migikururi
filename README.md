## ダウンロード・インストール
･ [Releases](https://github.com/nakanikki/migikururi/releases) から最新の `MigiKururi.exe` をダウンロード。起動。

･ ポータブル仕様です。設定ファイルは同列フォルダに作られます。作れない場合はAppDataに作ります。

･ アンインストールはファイル削除
